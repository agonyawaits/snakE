# Snake

## Overview
Snake game implementation in C++. Rendering is done using [ncurses](https://en.wikipedia.org/wiki/Ncurses) library.

## Build

### macOS
```console
$ brew install pkg-config ncurses
$ make
```

### Debian
```console
$ sudo apt install pkg-config libncurses5-dev libncursesw5-dev
$ make
````
