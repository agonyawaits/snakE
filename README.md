c++ object-oriented terminal snake
-------------------------------------------
<img src="https://github.com/agonyawaits/snakE/blob/redesign/snake.png" width="580" height="240">

To play the game:
1) Install ncurses lib (google)
2) Clone repository
```bash
$ git clone https://github.com/agonyawaits/snakE
```
3) Build and run:
```bash
$ cd snakE/ && ./build
$ ./bin/snake
```
