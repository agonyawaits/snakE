c++ object-oriented terminal snake
-------------------------------------------
<img src="./snake.png" width="420" height="260">

To play the game:
1) Install ncurses lib (google)
2) Clone repository
```bash
$ git clone https://github.com/agonyawaits/snakE
```
3) Build and run:
```bash
$ cd snakE/ && ./build
$ ./bin/snake
```
