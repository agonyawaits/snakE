Snake game written on C++ for Unix Terminal
-------------------------------------------
To play the game:
1) Install ncurses
```bash
$ brew install ncurses
```
2) Clone repository
```bash
$ git clone https://github.com/agonyawaits/snake
```
3) Compile and run:
```bash
$ g++ Apple.cpp Desk.cpp Game.cpp Snake.cpp main.cpp -lncurses -std=c++2a
$ ./a.out
```
